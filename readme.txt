== Description ==

This [WP Alert Bars](http://www.hossnimubarak.com) plugin allow users to display a notification/alert box in a Post or Page multiple times. Use shortcode: [wp_alert_bars name="success"]

**Plugin Features:**

* Easy to install
* Fully integrated in WordPress
* Unlimited alert/notification box/bar creation and display
* Easy to use
* Display on both Post and Page
* Shortcoded: [wp_alert_bars name="success"] or [wp_alert_bars name="success" desc="Write Some Description"]

== Installation ==

Install WP Alert Bars as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search WP Alert Bars
3. Click to install
4. Activate the plugin
5. Setup the proper settings in WP Alert Bars menu

**Install by upload plugin:**

1. Download the latest version of WP Alert Bars (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded wp-alert-bars.zip and click on "Install Now" button
6. Activate the plugin
